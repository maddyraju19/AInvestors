Welcome to AInvestors, a platform built for traders eager to level-up their strategies.
This platform takes your strategy as an input in text format
and uses machine learning to classify it into a trading strategy class.
Then it uses the strategy it identified plus some custom parameters
to backtest itself such that you can see the historical results of
your strategy if you were to implement it now!
